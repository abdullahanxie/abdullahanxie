## I build systems that turn intent into form.

My work lives where **vision models**, **coding agents**, **parametric CAD**, and
**product design** exchange materials. The recurring question behind my projects
is simple:

> Can software turn an ambiguous human idea into something spatial, editable, and real?

### Current engineering coordinates

| Signal | What I am building |
|---|---|
| **Pixels → Parts** | Sketch-to-3D, picture-to-CAD, and image-guided geometry |
| **Language → Structure** | Agent-native tools that generate and edit structured 3D assets |
| **Geometry → Products** | Catalog pipelines, ring design systems, and manufacturing-ready outputs |

### Workshop

```text
human intent
    └── sketch / image / language
            └── vision + coding agents

                            └── editable, physical object
```
